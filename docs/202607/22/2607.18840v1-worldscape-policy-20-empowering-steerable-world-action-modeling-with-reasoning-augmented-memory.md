## 摘要
世界动作模型（WAMs）通过联合建模视觉状态转换与机器人动作，为机器人操作提供了一种有前景的范式。然而，现有WAMs受限于有限的时间上下文、粗糙的回合级语言监督以及以文本为主的条件控制，这阻碍了任务进度跟踪与细粒度语言-视频-动作对齐，同时限制了视觉上下文推理和跨实体迁移。在本文中，我们提出了WorldScape Policy 2.0，一种具有推理增强长短期记忆的可控世界动作模型。其因果短期视觉记忆提供最近的观测作为DiT预填充，以保留局部交互动态；而其长短期事件记忆将历史VLM输出组织为全局历史、局部活跃和事件边界表示，用于进度感知检索。检索到的历史增强了感知和自回归生成的规划令牌，为自主规划提供隐式子目标条件；语义强制进一步将事件级指令语义转移到这一潜在规划路径中。为了建立细粒度多模态可控性，我们构建了ManipEvent-5M，一个事件赋能的具身预训练数据集，包含近500万个事件片段，对齐了动作轨迹、回合级任务指令、片段级子任务描述、目标图像和视频演示。这些设计为高层指令的自主规划和细粒度文本、目标图像或视频上下文提示的可控执行提供了统一接口。在仿真和真实世界平台上的实验表明，其在长时域自主规划、细粒度指令跟随和上下文适应方面具有优越能力。

## Abstract
World Action Models (WAMs) offer a promising paradigm for robotic manipulation by jointly modeling visual state transitions and robot actions. However, existing WAMs are constrained by limited temporal context, coarse episode-level language supervision, and predominantly text-only conditioning, which hinder task-progress tracking and fine-grained language-video-action grounding while limiting visual-context reasoning and cross-embodiment transfer. In this paper, we introduce WorldScape Policy 2.0, a controllable WAM with reasoning-augmented long short-term memory. Its causal short-term visual memory supplies recent observations as DiT prefill to preserve local interaction dynamics, while its long short-term event memory organizes historical VLM outputs into global-history, local-active, and event-boundary representations for progress-aware retrieval. The retrieved history augments perception and autoregressively generated planning tokens, yielding an implicit subgoal condition for autonomous planning; semantic forcing further transfers event-level instruction semantics into this latent planning pathway. To establish fine-grained multimodal controllability, we construct ManipEvent-5M, an event-grounded embodied pretraining dataset containing nearly 5 million event segments with aligned action trajectories, episode-level task instructions, segment-level subtask captions, goal images, and video demonstrations. These designs provide a unified interface for autonomous planning from high-level instructions and controllable execution from fine-grained text, goal-image, or video-context prompts. Experiments in both simulation and real-world platforms demonstrate superior capabilities in long-horizon autonomous planning, fine-grained instruction following and in-context adaptation.

### 研究动机

- **粗粒度的语言-视频-动作对齐**：同一回合的不同片段共享相同的任务级指令，为多样化的原子动作提供了弱语言监督，模型容易依赖视觉捷径而非真正学习事件级意图与状态转换的对应关系。
- **受限的交互接口**：仅支持文本条件控制，无法利用目标图像、演示视频等视觉提示，限制了视觉上下文推理和跨实体迁移能力。

### 整体含义
本文旨在将WAM从被动的短时预测模型转变为可控的长时策略。核心洞察是：长时可控性需要两种互补的时间上下文——用于推理任务进度的**语义事件记忆**和用于保留局部交互动态的**帧级视觉记忆**。基于此，作者提出**WorldScape Policy 2.0**，一种结合推理增强长短期记忆的可控WAM，同时构建了大规模事件级数据集**ManipEvent-5M**，支持从高层指令到细粒度多模态提示的统一接口。

## 2. 方法论文方法论

### 核心思想
- 将记忆分为两个互补层级：
  - **短时视觉记忆**：保留最近几帧的VAE潜在表示，作为DiT的因果预填充，保持局部交互动态。
  - **长短期事件记忆**：通过VLM分支组织历史片段为全局历史、局部活跃和事件边界三种表示，实现进度感知检索。
- **检索增强推理**：将检索到的事件记忆与当前感知和自回归规划令牌融合，得到隐式**子目标潜在条件**，用于自主规划。
- **语义强制**：在训练时将事件级子目标描述（T5嵌入）作为语义目标，迫使VLM生成的推理潜在与子目标语义对齐，从而将可控制的细粒度语言理解迁移到自主规划路径中。
- **统一交互接口**：支持多种控制模式：
  - **自主规划**：仅输入高层任务指令，模型通过事件记忆推理出当前子目标并生成动作。
  - **细粒度指令跟随**：直接使用T5编码的片段级子目标描述作为DiT交叉注意力条件。
  - **视觉提示控制**：目标图像或演示视频作为持久视觉预填充，支持目标条件和上下文学习。

### 关键技术细节
- **模型架构**：
  - 视频/动作共享一个因果DiT（基于Wan2.2-5B初始化）。
  - VLM推理分支使用轻量级Qwen3-VL-4B，处理头视图观测并生成4个规划令牌。
  - 每个实体有独立的动作编码器/解码器，但共享DiT主干。
- **动作表示**：相对增量位姿（3D位置+6D连续旋转）+ 绝对夹爪命令，共10D/臂，双臂20D。
- **事件记忆构建**：
  - 历史VLM输出（感知令牌+规划令牌）压缩为压缩全历史库（通过注意力池化转化为固定数量gist令牌）。
  - 显式构建三种记忆视图：全局历史、局部活跃（最近S_e个片段）、事件边界（通过余弦相似度变化检测选择S_b个最高变化索引）。
  - 检索后通过门控融合（可学习sigmoid门控）注入当前令牌。
- **语义强制损失**：使用固定T5目标的归一化表示与VLM推理潜在之间的余弦相似度（停止梯度），训练投影器对齐。
- **训练目标**：联合流匹配损失（动作+视频潜在） + 语义强制损失（λ_s=0.001）。

### 算法流程（三阶段课程学习）
1. **阶段1：事件级多模态WAM预训练**：基于ManipEvent-5M，使用细粒度子目标描述、目标图像、视频演示等条件，优化动作+视频流匹配损失，不启用VLM事件记忆。
2. **阶段2：记忆增强的中训练**：冻结阶段1权重，引入VLM推理分支和事件记忆，使用高层指令作为输入，通过语义强制损失（L_sem）对齐推理潜在与子目标语义。继续优化流匹配损失。
3. **阶段3：下游交互后训练**：根据任务需求选择控制模式，在具体机器人平台微调，适应交互风格。

## 7. 优点

1. **创新的记忆架构**：首次将长短期事件记忆（推理级）与短期视觉记忆（动态级）显式解耦并联合使用，实现进度感知的隐式规划，区别于单纯全历史或滑动窗口的方法。
2. **大规模事件级数据集**：ManipEvent-5M涵盖多种数据源（人、机器人、仿真）、近500万片段，且附带细粒度子目标描述、目标图像、视频演示，为可控WAM训练提供坚实数据基础。
3. **统一的多模态控制接口**：一个模型同时支持自主规划、细粒度指令、目标图像、视频上下文，无需为不同任务切换模型或额外训练。
4. **语义强制技术**：将VLM推理潜在与T5语义对齐，使自主规划路径继承预训练的语言理解能力，无需显式子目标标注即可实现隐式推理。
5. **三阶段课程学习**：从可控预训练到记忆推理再到下游适应，逐步引入复杂度，避免冷启动问题，有效迁移知识。
6. **实验结果全面且优秀**：在仿真和真实平台均大幅超越当前SOTA，跨实体迁移能力尤显优势。

4. **事件边界检测依赖VLM输出变化**：通过余弦相似度选择事件边界，虽无监督，但可能受VLM生成质量影响，在噪声数据下误检或漏检。

6. **消融实验仅用clean训练**：虽然说明不与主结果直接比较，但clean-only设置下的绝对成功率较低（完整模型仅47.89%均值），可能无法充分体现记忆组件在随机化条件下的增强效果（随机化条件下STM/LTM提升幅度更大）。
7. **未进行统计显著性检验**：未报告置信区间或p值，无法判断结果差异的统计可靠性。
8. **对细粒度指令的依赖**：在自主规划模式下，隐式子目标推理依赖于训练时的语义强制，若任务指令过于模糊或环境出现未见过的子目标，可能推理失败。
