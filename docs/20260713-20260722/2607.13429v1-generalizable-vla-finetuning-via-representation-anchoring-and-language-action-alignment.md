## 摘要
通过行为克隆（BC）在机器人演示上微调预训练的视觉语言模型（VLM）已成为视觉-语言-动作（VLA）策略的标准方法。然而，BC微调会逐步覆盖支持视觉和语义泛化的预训练表示。常见的补救措施——在网络图像-文本数据上的联合训练——并不能防止这种情况；它将语言和动作损失应用于不同的观测，导致VLA出现语言-动作不对齐，而标准的操作基准测试无法暴露这一问题。我们提出Anchor-Align，它通过两个目标增强BC：视觉-语言锚定从冻结的VLM副本中蒸馏逐层表示以防止这种漂移，而语言-动作对齐将每个动作目标转换为离散的运动方向标签，并在同一机器人观测上联合训练语言和动作预测。在物理xArm7机器人上，跨两种广泛使用的VLA架构，Anchor-Align在真实机器人成功率上分别从28%提升至54%和从37%提升至60%。在仿真规模实验中，我们在LIBERO-PRO、LIBERO-Plus和CALVIN上分别展示了在OOD扰动、感知鲁棒性和长程控制方面的一致改进，这表明保留预训练表示与有效的动作学习并非根本对立。项目页面：anchoralignvla.github.io

## Abstract
Finetuning a pretrained vision-language model (VLM) on robot demonstrations via behavior cloning (BC) has become the standard recipe for vision-language-action (VLA) policies. However, BC finetuning progressively overwrites the pretrained representations that support visual and semantic generalization. Co-training on web image-text data, a common remedy, does not prevent this; it applies language and action losses to separate observations, leaving VLAs with language-action misalignment that standard manipulation benchmarks do not expose. We propose Anchor-Align, which augments BC with two objectives: Vision-Language Anchoring distills layer-wise representations from a frozen VLM copy to prevent this drift, while Language-Action Alignment converts each action target into a discrete motion-direction label and jointly trains language and action prediction on the same robot observation. On a physical xArm7 robot, across two widely used VLA architectures, Anchor-Align improves real-robot success on both (28% to 54% and 37% to 60%). At scale in simulation, we demonstrate consistent improvements on OOD perturbations, perceptual robustness, and long-horizon control across LIBERO-PRO, LIBERO-Plus, and CALVIN, respectively, suggesting that preserving pretrained representations and effective action learning are not fundamentally at odds.   - **语言-动作不对齐**：常见的补救措施（如联合训练网络图像-文本数据）并不能防止表示覆盖，反而因为对不同的观测分别施加语言损失和动作损失，导致语言与动作之间出现不对齐。标准的操作基准测试无法暴露这一问题。
- **研究意义**：提出一种新方法，在保留预训练表示的同时实现有效的动作学习，证明这两者并非根本对立。

## 2. 论文提出的方法论

### 核心思想：Anchor-Align

通过两个辅助目标增强 BC 微调：

- **视觉-语言锚定（Vision-Language Anchoring）**：
  - 从冻结的 VLM 副本中蒸馏逐层表示，防止微调过程中预训练表示的漂移。
  - 具体：在训练时，VLA 模型与一个冻结的预训练 VLM（如 CLIP）共享输入，将 VLA 每层的输出对齐到冻结 VLM 对应层的输出（可能使用蒸馏损失，如 MSE 或 KL 散度）。

- **语言-动作对齐（Language-Action Alignment）**：
  - 将每个连续动作目标转换为离散的运动方向标签（例如“向左”、“向右”、“向前”、“抓取”等）。
  - 在同一个机器人观测上，同时训练语言预测（如预测动作对应的方向词）和动作预测，使语言与动作在表示空间中对齐。
  - 这样，语言损失和动作损失施加于同一观测，避免了传统联合训练中的不对齐。

### 技术细节（文字描述）

1. 使用预训练的 VLM（例如 CLIP 或 BLIP）作为视觉编码器。
2. 动作头（action head）通常是一个 MLP，输出连续动作（如关节角度或末端执行器位姿）。
3. Anchor 模块：复制一个冻结的预训练 VLM，在训练时对 VLA 模型的每一层（或特定层）施加蒸馏损失，迫使 VLA 模型的表示接近冻结 VLM 的表示。
4. Align 模块：将动作标签离散化为 K 个方向类别（如 8 个方向 + 1 个抓取），动作头不仅输出连续动作，还输出离散方向 logits，使用交叉熵损失训练方向预测；同时保持原始连续动作回归损失。
5. 总损失 = BC 损失 + 锚定蒸馏损失 + 方向对齐损失。

### 算法流程（文字说明）

1. 输入机器人观测图像 + 自然语言指令。
2. 通过 VLA 模型提取特征，并输出动作（连续）和方向类别（离散）。
3. 冻结的 VLM 副本输出参考特征。
4. 计算三部分损失并反向传播更新 VLA 模型（冻结副本不更新）。
