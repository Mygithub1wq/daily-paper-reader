## 摘要
类似于人类顺序学习新任务的自然能力，配备视觉-语言-动作（VLA）模型的机器人在开放世界环境中部署时应具备终身学习能力，以学习新任务。然而，最近提出的大多数终身学习模型旨在有效学习当前任务（可塑性）或保持对先前任务的高准确性（稳定性），而在机器人操作模型中，可塑性与稳定性的权衡仍未得到解决。为应对这一基本挑战，我们提出了一种用于机器人操作的缓存高效终身视觉-语言-动作学习框架（即LifelongVLA），该框架通过双时间尺度适应机制缓解可塑性与稳定性的权衡，同时通过缓存高效的重放策略实现低成本的机器人部署。更具体地说，我们提出了一种双时间尺度LoRA门控模块，将VLA适应分解为两条轻量级路径：用于可塑性的短期适配器和用于稳定巩固的长期适配器。这些路径通过任务感知门控进行集成，从而实现对可塑性与稳定性权衡的显式控制。在技能重放阶段，提出了一种缓存高效的随机重放策略，以在不存储完整轨迹的情况下保留更平衡的保留信号。最后，实验表明，LifelongVLA优于现有基线，展示了高效的技能扩展、对已学习操作行为的稳健保留以及在实际xArm机器人上部署时对重新训练的依赖减少。

## Abstract
Similar to the natural capabilities of humans to sequentially learn new tasks, robots with Vision-Language-Action (VLA) models should possess lifelong learning ability to learn a new task when deployed in open-world environments. However, most recently proposed lifelong learning models aim to effectively learn the current task (plasticity) or maintain high accuracy on previous tasks (stability), while the plasticity-stability trade-off remains largely unsolved in robotic manipulation models. To address this fundamental challenge, we propose a cache-efficient lifelong Vision-Language-Action learning framework for robotic manipulation (i.e., LifelongVLA), which alleviates the plasticity-stability trade-off with a dual-timescale adaptation mechanism while achieving low-cost robotic deployment with a cache-efficient replay strategy. More concretely, we propose a dual-timescale LoRA gating module to decompose VLA adaptation into two lightweight pathways: a short-term adapter for plasticity and a long-term adapter for stable consolidation. These pathways are integrated via a task-aware gate, enabling explicit control of the plasticity-stability trade-off. In the skill replay phase, a cache-efficient stochastic replay strategy is proposed to preserve more balanced retention signals without full-trajectory storage. Finally, experiments show that LifelongVLA outperforms existing baselines, demonstrating efficient skill expansion, robust retention of learned manipulation behaviors, and reduced reliance on retraining for real-world deployment on an xArm robot.

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：视觉-语言-动作（VLA）模型将视觉-语言预训练与机器人动作生成结合，已成为语言条件机器人操作的关键范式。现有VLA政策大多采用静态离线训练范式——在固定技能集上训练一次后部署，不再更新。
- **核心问题**：真实世界的机器人部署中，新物体、新指令、新场景布局和本体动作分布会持续出现，因此机器人需要像人类一样具备**终身学习能力**，在不遗忘已学技能的前提下不断获取新技能。然而，直接顺序学习会导致**灾难性遗忘**；而保存全部历史数据并反复重训会带来巨大的存储和计算开销。终身学习中的**可塑性-稳定性权衡**（plasticity-stability trade-off）在机器人操作模型中仍未得到有效解决。
- **研究含义**：本文旨在设计一个缓存高效的终身VLA学习框架，使机器人能够持续获取新操作技能，同时保持对旧技能的高精度，并减少重训依赖，推动机器人向类人物理智能迈进。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
LifelongVLA通过两个核心模块解决可塑性-稳定性权衡和重放开销问题：
1. **双时间尺度LoRA门控模块（Dual-Timescale LoRA Gating）**：将VLA适应分解为短期和长期两条LoRA路径，通过任务感知门控在权重级别动态组合，实现可塑性与稳定性的显式控制。
2. **缓存高效随机重放模块（Cache-Efficient Stochastic Replay）**：仅存储少量选样的停止梯度前缀令牌和紧凑的状态-动作监督，在重放时重新采样扩散变量并重新计算后缀令牌，以提供丰富的保留信号，避免完整轨迹存储和重复编码。

### 关键技术细节

#### 双时间尺度LoRA门控
- 对每一层 ℓ，定义短期更新 ΔW^{ℓ}_{sh,t} 和长期更新 ΔW^{ℓ}_{lg,t}，均由低秩分解 A^{ℓ}B^{ℓ} 构成。
- 优化方式不同：短期路径仅由当前任务损失 L_new 更新（高学习率 η_sh），保证高可塑性；长期路径仅由重放损失和蒸馏损失更新（低学习率 η_lg），缓慢演进以巩固旧技能。
- 门控上下文 c 从冻结前缀特征池化得到（使用 stopgrad），避免循环依赖。
- 门控函数 α^{ℓ}_t(c) = σ(W^ℓ_g ϕ^ℓ(c) + b^ℓ_g) 输出 [0,1] 值，控制长期路径的贡献。
- 权重级组合：Δ̃W^{ℓ}_t(c) = (1-α)ΔW^{ℓ}_{sh} + α ΔW^{ℓ}_{lg}，最终权重 W^{ℓ}_0 + γ Δ̃W^{ℓ}_t(c)。一次前向通过即可完成。

#### 缓存高效随机重放
- 每个新任务随机选择少量样本（缓存配额 c_t=50），存储其停止梯度前缀令牌、机器人状态和动作标签，不存储原始图像、语言令牌、扩散时间步等。
- 重放时，重新采样扩散时间步 τ' 和噪声 ϵ'，构造带噪动作 ý^{rep}；使用当前模型重新计算后缀令牌 h^{suf,rep}_{j,t}，然后通过动作解码器预测。
- 当前样本和重放样本打乱混合，使用掩码分离损失 L_new 和 L_rep。重放样本还使用前一个任务的冻结模型快照作为教师计算蒸馏损失 L_dist。
- 总损失：L_t = L_new + λ L_rep + β L_dist，其中 λ=1.0, β=0.1。
