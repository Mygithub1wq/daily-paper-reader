## 摘要
视觉语言导航（VLN）要求具身智能体理解自然语言指令，并从时序视觉观测中预测动作。将多模态大语言模型适配至VLN需要视觉-语言对齐、紧凑的时序输入、动作空间接地以及目标硬件上的稳定训练。本技术报告介绍PGN（盘古导航器），一个基于OpenPangu-7B构建的离线VLN动作预测系统。训练分为两个阶段。第一阶段，PGMM通过训练Q-Former和两层MLP投影器，将冻结的EVA-ViT-G/14视觉编码器与冻结的语言主干对齐。第二阶段，PGN利用五观测窗口、依赖epoch的时序采样以及推理-动作输出格式，将对齐后的模型适配至专家导航轨迹；此阶段冻结对齐的视觉通路，并更新三种结构词元嵌入和LoRA适配器。实现结合了混合精度计算、选择性FP32计算以及八个Ascend 910B NPU上的DeepSpeed ZeRO-2。在500条保留的专家轨迹上进行教师强制开环评估时，V9报告了62.29%的归一化动作匹配（NAM）和100.00%的非空率（NER）。这些指标量化了离线专家动作对齐，而非闭环导航成功率；评估误差累积、路径效率和目标完成度留待未来工作。

## Abstract
Vision-Language Navigation (VLN) requires an embodied agent to interpret a natural-language instruction and predict actions from temporally ordered visual observations. Adapting a multimodal large language model to VLN requires visual-language alignment, compact temporal inputs, action-space grounding, and stable training on the target hardware. This technical report presents PGN (Pangu Navigator), an offline VLN action-prediction system built on OpenPangu-7B. Training proceeds in two stages. First, PGMM aligns a frozen EVA-ViT-G/14 vision encoder with the frozen language backbone by training a Q-Former and a two-layer MLP projector. Second, PGN adapts the aligned model to expert navigation trajectories using five-observation windows, epoch-dependent temporal sampling, and a reasoning-then-action output format; this stage freezes the aligned visual pathway and updates three structural-token embeddings and LoRA adapters. The implementation combines mixed-precision computation, selective FP32 computation, and DeepSpeed ZeRO-2 on eight Ascend 910B NPUs. Under teacher-forced, open-loop evaluation on 500 held-out expert trajectories, V9 reports a 62.29% Normalized Action Match (NAM) and a 100.00% Non-empty Rate (NER). These metrics quantify offline expert-action alignment rather than closed-loop navigation success; evaluating error accumulation, path efficiency, and goal completion remains future work.

- **研究问题**：视觉语言导航（VLN）要求具身智能体理解自然语言指令，并从时序视觉观测中预测动作。将通用多模态大语言模型适配到连续导航任务面临四个关键挑战：视觉-语言对齐、紧凑时序输入表示、将模型输出接地到有限动作空间，以及在特定硬件（如Ascend NPU）上实现稳定训练。
- **应用场景**：室内机器人导航，智能体需根据指令（如“走过厨房，在花瓶旁停下”）执行前进、左转、右转、停止等动作。
- **现有工作局限**：早期VLN使用离散导航图（R2R），后续VLN-CE扩展到连续环境；但通用MLLM直接用于导航时缺乏专用对齐和适配方案。该论文提出PGN系统，基于OpenPangu-7B语言骨干，通过两阶段训练实现离线动作预测。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：采用两阶段训练策略：
  - **阶段1（视觉-语言对齐）**：冻结EVA-ViT-G/14视觉编码器和OpenPangu-7B语言骨干，仅训练Q-Former（32个可学习查询令牌）和两层MLP投影器，将图像特征映射到语言模型嵌入空间。训练数据为图像-字幕对（MS COCO、Visual Genome）。
  - **阶段2（导航适配）**：冻结对齐后的视觉通路（EVA-ViT, Q-Former, MLP投影器）和基础语言模型，仅更新三个结构令牌嵌入（图像开始、图像结束、帧边界）和LoRA适配器（注入语言模型）。使用专家导航轨迹进行监督学习。
- **输入格式**：每个时间步输入一个固定长度5的观测窗口：当前观测加上前4个历史观测，时序顺序保留。观测经过共享视觉通路后，与自然语言指令和结构令牌拼接形成自回归上下文。
- **输出格式**：采用“推理文本+规范动作”的格式，即先输出推理原因，再输出标准化动作（forward, left, right, stop）。
- **时序采样策略**：在阶段2训练中，根据epoch轮换采样起始索引（公式5），并使用动作感知采样以平衡不同动作类别的出现概率（三个阶段参数分别为(s=4, p=0.1), (s=6, p=0.6), (s=8, p=0.2)）。
- **优化与硬件实现**：混合精度计算，但部分FP16注意力掩码操作因数值溢出改用FP32；使用DeepSpeed ZeRO-2在8个Ascend 910B NPU上分布式训练。

## 3. 实验设计

- **数据集**：

  - 验证集：500条保留的专家轨迹。
- **评估协议**：教师强制（teacher-forced）、开环（open-loop）评估。模型接收来自专家轨迹的固定五观测窗口（包含历史观测和当前观测），预测动作，但不执行动作以获取下一观测。因此评估的是离线专家动作对齐，不包含闭环误差累积。

## 6. 主要结论与发现

- 跨版本趋势：从V5到V6，NER从33.98%提升到99.17%（输出格式改进），但NAM略有下降（25.40%→23.18%）。V8和V9实现NAM约62.3%和NER 100.00%，表明引入推理-动作输出、身份提示、时序采样和动作感知采样后，离线动作匹配显著改善。

- 论文总结：PGN证明了将OpenPangu-7B适配到离线专家动作对齐的可行性，但未建立闭环导航成功。
