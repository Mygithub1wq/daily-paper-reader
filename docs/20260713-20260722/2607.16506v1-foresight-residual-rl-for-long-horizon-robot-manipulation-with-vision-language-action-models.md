## 摘要
视觉-语言-动作（VLA）策略提供了强大的通用操控先验，但在高精度、接触密集的装配任务中常因长程信用分配和子任务耦合而失败：对于当前技能在几何上成功的状态，可能对后续技能是脆弱的。我们展示了在冻结的VLA基础策略上进行残差强化学习（RL）时的这种失败模式：恒定的稀疏成功奖励虽然单独改善了每个子任务，但当技能串联时几乎没有增益，因为终端状态质量未受控制。我们提出远见残差强化学习（Foresight Residual RL），通过用离线估计的远见值——即当前子任务终端状态下后续子任务成功的条件概率——增强每个子任务的稀疏成功奖励，来优化交接质量。具体地，我们（i）从基础策略终端状态的图像训练一个视觉远见预测器，使用下游回滚统计数据作为标签；（ii）通过反向远见归纳训练残差策略，将预测器输出作为奖励乘数。在Isaac Gym中一个三阶段的扳手式螺母拧紧装配任务（抓取、移动-插入、旋转）中，我们的方法实现了85.6%的全任务成功率，优于标准子任务残差RL（54.5%）和VLA基线，同时保持每子任务成功率不变。这些结果强调，改善长程性能需要塑造每个子任务产生的成功状态，而不仅仅是是否成功。

## Abstract
Vision-Language-Action (VLA) policies offer strong general-purpose manipulation priors, but often fail on tight-tolerance, contact-rich assembly due to long-horizon credit assignment and subtask coupling: a state that is geometrically successful for the current skill can be brittle for downstream skills. We show this failure mode in residual reinforcement learning (RL) over a frozen VLA base policy: constant sparse success rewards improve each subtask in isolation yet yield little or no gain when skills are chained, because terminal state quality is uncontrolled. We propose Foresight Residual RL, which optimizes handoff quality by augmenting each subtask's sparse success reward with an offline-estimated foresight value -- the probability of future subtask success conditioned on the terminal state of the current subtask. Concretely, we (i) train a visual foresight predictor from images of terminal states of the base policy, labeled using downstream rollout statistics, and (ii) train residual policies via backward foresight induction, using the predictor output as a reward multiplier. On a three-phase wrench-based nut-tightening assembly task in Isaac Gym (grasp, move-insert, rotate), our method achieves 85.6% full-task success, outperforming standard subtask residual RL (54.5%) and VLA baselines, while leaving per-subtask success unchanged. These results highlight that improving long-horizon performance requires shaping which successful states are produced at each sub-task, not only whether success occurs.

论文聚焦于**长程、接触密集的机器人操控任务**（如工业装配、工具使用），这类任务通常被分解为多个子任务（sub-tasks）并分别训练策略，然后串联执行。然而，现有方法（包括VLA基础策略和残差强化学习）存在一个关键失败模式：**子任务之间的因果耦合**——当前子任务的终端状态（终端状态）即使满足几何成功条件，也可能对下游子任务非常脆弱（例如，抓取扳手的姿态虽然成功，但导致后续插入失败）。标准残差RL在每个子任务中使用恒定的稀疏成功奖励，虽然能独立提升各子任务的成功率，但串联时全任务成功率几乎无提升，因为**终端状态的质量未被控制**。论文指出，提升长程性能的核心不在于是否让子任务成功，而在于**产生什么样的成功状态**——即优化子任务终端状态的“远见价值”（foresight value）。

整体含义：**将子任务级优化与全局目标对齐，通过离线学习的远见预测器来引导残差策略产生更利于下游成功的终端状态，从而打破子任务耦合瓶颈。**

## 2. 论文提出的方法论

### 核心思想
提出**Foresight Residual RL**框架，核心是：
- 为每个子任务边界（如抓取→插入）训练一个**视觉远见预测器**（visual foresight predictor），该预测器从当前子任务终端状态的图像中估计下游子任务成功的概率。
- 在残差策略训练中，将预测器的输出作为**奖励乘数**，与子任务成功的稀疏奖励相乘，形成**远见调节的奖励**：`r_t = β_k(s_{t+1}) · p_ϕ(o_{t+1})`，从而引导策略产生具有高远见价值的终端状态。
- 采用**反向远见归纳**（Backward Foresight Induction）训练顺序：从最后一个子任务反向训练，每个子任务训练时下游策略已固定，保证远见估计的平稳性。

### 关键技术细节
1. **残差策略设计**：动作 `a_t = π_k^0(o_t) + α · π_k^res(o_t)`，α << 1，保证残差动作幅度小，基础策略行为基本保留。
2. **远见预测器训练（Algorithm 2）**：
   - 用基础策略π_k^0收集N个成功的终端状态；
   - 对每个终端状态，执行K_rep次下游策略π_{k+1}的独立回滚，记录成功次数κ_i；
   - 以binomial negative log-likelihood为损失函数，训练图像编码器（DINOv2-S/14 + 自注意力 + 线性层 + sigmoid头）输出概率p_ϕ(o)。
3. **奖励设计**：仅使用上述远见调节奖励，无其他外部奖励或势函数。
4. **训练流程**：PPO算法，128个并行环境，先进行10次critic预热，再联合训练300次迭代。

### 公式和算法流程（文字说明）
- 定义远见值函数 V_{k+1:K}(s) = 从状态s开始执行下游策略π_{k+1:K}的期望最终阶段成功折扣回报。
- 局部远见目标：V_k^◦(s) = E[∑γ^t β_k(s_{t+1}) V_{k+1:K}(s_{t+1})]。
- 算法1：反向远见驱动学习：先训练最后子任务（K），然后依次向前，每个阶段先训练预测器（Algorithm 2），再用远见奖励训练残差策略。

## 6. 论文的主要结论与发现

1. **标准残差RL的局限性**：虽然能显著提升单子任务成功率（如插入从45.7%→92.2%），但全任务成功率仅54.5%，与端到端π0的54.9%相近。原因在于残差策略优化了成功率却牺牲了终端状态的质量（远见预测器得分从0.64降至0.32）。
2. **远见校正的有效性**：本文方法将全任务成功率提升至85.6%，较标准残差RL提升31.1个百分点。主要增益集中在抓取→插入边界（+31.8%），与该边界预测器的判别能力更强（准确率86.2% vs 基线73.9%）一致。
3. **终端状态质量改善**：远见校正策略的终端状态平均预测器得分显著高于标准残差RL（抓取终端0.73 vs 0.32，插入终端0.91 vs 0.86），表明其产生了更利于下游成功的状态。
4. **定性验证**：可视化表明，即使两个方法都满足子任务成功谓词，标准残差RL的终端状态（如抓取角度不佳）在后续阶段会失败，而远见校正状态具有更好的对齐和稳定性。
